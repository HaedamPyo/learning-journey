# 2과목 - 머신러닝

## 회고
머신러닝에 대한 기초 이론을 학습하고,
전체 파이프라인 순서를 반복 훈련하며 흐름을 익혔다.\
문제정의부터 결과파일 생성까지 손으로 직접 짜보면서\
"모델을 잘 만드는 것"보다 "전체 흐름을 완성하는 것"이 먼저라는 감각을 얻었다.

## 학습 내용 정리

### CHAP1. 머신러닝 기초

### 1. 머신러닝이란?
데이터를 학습 → 패턴 포착 → 의사결정에 활용하는 인공지능 분야

### 2. 머신러닝의 분류

#### 2-1. 지도학습 (Supervised Learning)
정답(label)이 정해져 있는 상태에서 학습하는 방식\
데이터가 **feature(입력값)** + **label(정답)** 구조로 구성됨\
**목적에 따른 분류**
| 유형 | 설명 |
|---|---|
| 회귀 (Regression) | 결과값이 연속된 수(수치)로 나옴 |
| 분류 - 이진 (Binary Classification) | 결과값이 두 그룹으로 나뉨 |
| 분류 - 다중 (Multiclass Classification) | 결과값이 세 개 이상(N개)의 그룹으로 나뉨 |

#### 2-2. 비지도학습 (Unsupervised Learning)
주어진 정답이 없고, 숨겨진 구조나 패턴을 찾는 문제
**목적에 따른 분류**
| 유형 | 설명 |
|---|---|
| 군집화 (Clustering) | 정답 라벨 없이 데이터의 유사성을 기준으로 여러 그룹(군집)으로 나눔 |
| 이상치 탐지 (Anomaly/Outlier Detection) | 대부분의 데이터와 크게 다른 패턴을 보이는 데이터를 찾아냄 |

#### 2-3. 강화학습 (Reinforcement Learning)
- 환경(Environment) & 보상(Reward) 기반으로 학습.

### 3. 지도 학습의 평가지표

#### 3-1. 분류
- **정확도(Accuracy)** : 전체 데이터 중 올바른 예측 비율
- **정밀도(Precision)** : 양성으로 예측한 것 중 맞은 예측 비율
- **재현율(Recall)** : 실제 양성인 데이터 중 올바른 예측(민감도)
- **F1-score** : 정밀도&재현율의 조화 평균 `2 × (정밀도 × 재현율) / (정밀도 + 재현율)`
- **ROC-AUC** : 임계값(confidence)을 바꿔가며 재현율과 위양성율의 관계를 그린 곡선(ROC)아래의 면적, 1에 가까울 수록 성능이 좋음

#### 3-2. 회귀
- **MSE**(Mean Squared Error) : 오차 제곱의 평균, 큰 오차에 대해 가중치를 부여함
- **MAE**(Mean Absolute Error) : 오차 절댓값의 평균
- **결정계수**(R²) : 모델의 설명(예측)력, 1에 가까울수록 설명력이 높음
- **RMSE**(Root Mean Squared Error) : MSE의 제곱근, 큰 오차에 대한 가중치 부여 + 실제값과 같은 단위
- **MSLE**(Mean Squared Log Error) : 로그 씌운 오차의 제곱의 평균, 큰 오차의 영향을 축소함
- **RMSLE**(Root Mean Squared Log Error) : MSLE의 제곱근, 큰 오차의 영향 축소 + 실제 값과 같은 단위
- **MAPE**(Mean Absolute Percentage Error) : 예측값이 실제 값에서 평균적으로 몇 % 벗어났는지(오차율)

#### 3. EDA
